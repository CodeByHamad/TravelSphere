# TravelSphere - Smart Travel Planner & Destination Explorer

Overview
TravelSphere is a Java-based desktop application designed to help users explore destinations, manage travel plans, view tour packages, and maintain travel history.

## Features
- User Registration & Login
- Profile Management
- Destination Explorer
- Tour Package Management
- Travel History Tracking
- User Gallery
- MySQL Database Integration
- Secure Authentication

## Technologies Used
- Java
- Java Swing
- MySQL
- JDBC
- OOP Concepts

## Modules
### Authentication Module
- Login
- Registration
- Password Validation

### User Dashboard
- User Profile
- Account Settings

### Destination Module
- Browse Destinations
- Search Locations

### Tour Management
- View Packages
- Book Tours

### Travel History
- Previous Trips
- Saved Records

## Database
MySQL database used for storing:
- User Information
- Destinations
- Tour Packages
- Travel History
